# Install 

- `npm install --save react-big-calendar`
- Add import to main.jsx
  - `import "react-big-calendar/lib/css/react-big-calendar.css";`
- Must include a localizer library
  - Moment.js - `npm install moment`
  - Globalize.js
  - date-fns
  - day.js
